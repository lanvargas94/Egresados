# Assets Folder

Esta carpeta contiene los recursos estáticos de la aplicación.

## Logo de la Institución

El logo de CORHUILA debe guardarse aquí como:
- **Nombre del archivo**: `logo.png`
- **Formato**: PNG con fondo transparente
- **Tamaño recomendado**: Mínimo 560px de ancho (para alta calidad)
- **Ubicación**: `frontend/src/assets/logo.png`

El logo se mostrará automáticamente en la página de ingreso de la plataforma.
